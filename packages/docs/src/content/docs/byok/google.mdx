## Get a key

1. [ai.google.dev](https://ai.google.dev) → **Get API Key**
2. Create a key in a Google Cloud project

## Set it

```sh
basalt config set provider.google.api_key ...
```

## Models used

| Use | Model |
| --- | --- |
| Embeddings | `text-embedding-004` |
| Synthesis | `gemini-2.5-flash` |
